# TINapay NI CHaN

Static bakery site: `index.html`, `styles.css`, and `app.js`.

## Deploy on GitHub Pages

1. Push this repository to GitHub (see below).
2. On GitHub: **Settings → Pages**.
3. Under **Build and deployment**, set **Source** to **Deploy from a branch**.
4. Choose branch **main** (or **master**) and folder **/ (root)**.
5. Save. Your site will be at `https://<your-username>.github.io/<repo-name>/`.

If the site loads but assets fail, ensure `index.html` references `styles.css` and `app.js` with relative paths (they already do).
